
# Tocador de Rádio Online

Este e um projeto que eu sempre quiz desenvolver porem não fazia por não saber HTML e nem CSS, com o curso que estou fazendo agora da DevClub 
isso foi possivel, um curso que para mim esta abrindo um universo de possibilidades, curso extremamente recomendado. 
Esse site e para quem gosta de ouvir musicas como eu e ele esta disponivel aqui para quem quiser mudar e melhorar pois você pode colocar
a rádio que vc quiser nele sem problema algum lembrando que ele so funciona em HTML5.
Esse projeto e bem simples eu sei mas levou um tempinho para eu terminar pois tive que pesquisar muitos comandos para conseguir fazer do jeito que
eu queria, pesquisei no Google em vaias paginas, no proprio site do DevClub tem muito material para se pesquisar também eu usei as IAs treinadas em HTML e CSS
e a pesquisa resultou nesse site. São muitos comandos para se lembrar srsrsrsr, mas isso so com o tempo de desenvolvimento que conseguimos fixar essas
informações na cabeça.
